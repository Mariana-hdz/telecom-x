# Telecom X - Análisis de Evasión de Clientes (Churn)

## 📋 Descripción
Análisis exploratorio de datos (EDA) para identificar factores asociados a la evasión de clientes en Telecom X. El objetivo es comprender qué perfiles tienen mayor probabilidad de cancelar el servicio y proponer estrategias de retención.

## 📊 Resultados Clave
- **Tasa de evasión:** 26.54%
- **Factor más crítico:** Contrato mensual (42.7% evasión)
- **Método de pago de riesgo:** Electronic check (45.3% evasión)
- **Servicio con mayor evasión:** Fibra óptica (41.9% evasión)
